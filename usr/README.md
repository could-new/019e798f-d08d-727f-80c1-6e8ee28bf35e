# Mechanical Engineering Resume

A beautiful, interactive, cross-platform resume application built with Flutter, designed specifically to showcase mechanical engineering skills, experience, and projects.

## Features
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop viewports. Displays a modern two-column layout on wide screens and a sleek single-column scrollable interface on phones.
- **Professional Engineering Theme**: Styled with a dark blue and industrial orange color palette.
- **Comprehensive Sections**: 
  - Professional Summary
  - Experience (with detailed bullet points)
  - Education & Credentials
  - Notable Projects
  - Core Skills (SolidWorks, AutoCAD, ANSYS, GD&T)

## Tech Stack
- Flutter (Cross-platform UI toolkit)
- Dart

## Setup & Run Instructions
1. Ensure you have Flutter installed on your machine.
2. Clone or download this repository.
3. Open a terminal in the project root directory.
4. Run `flutter pub get` to fetch dependencies.
5. Run `flutter run` to launch the application on your preferred device (Web, Desktop, or Mobile).

## Deployment
This app can be deployed to the web, iOS, Android, macOS, Windows, and Linux. For web deployment, run `flutter build web` and host the generated `build/web` folder on platforms like Vercel, Firebase Hosting, or GitHub Pages.
